# Exoplanet Data Integration and Wrangling

## Project Overview

This project explores how stellar properties relate to the habitability of exoplanets using the Earth Similarity Index (ESI). Data was collected from multiple sources, cleaned, standardized, and merged into a single analytical dataset to investigate relationships between host star characteristics and potentially habitable planets.

## Research Question

How do stellar properties relate to Earth-like planets that are habitable or have a high Earth Similarity Index (ESI)?

## Data Sources

### Dataset 1: NASA Exoplanet Dataset

- Exoplanet characteristics
- Orbital information
- Planet classifications
- Discovery information

### Dataset 2: NASA Exoplanet Archive API

- Host star information
- Stellar temperature
- Stellar radius
- Stellar mass
- Stellar classifications

### Dataset 3: Habitable Worlds Catalog

- Potentially habitable exoplanets
- Earth Similarity Index (ESI)
- Planetary characteristics related to habitability

## Data Preparation

The project included:

- Data quality assessment
- Missing value handling
- Data type corrections
- Column standardization
- Dataset integration and merging
- Creation of a final analytical dataset

## Project Outcome

A unified dataset was created by integrating planetary, stellar, and habitability data from three separate sources. The resulting dataset can be used for further exploratory analysis and machine learning applications focused on exoplanet habitability.
